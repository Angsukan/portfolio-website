# Data Analyst Portfolio Website

Welcome to my portfolio website repository! This project showcases my skills, experience, and projects as a Data Analyst, utilizing a modern, premium dark-themed dashboard interface.

Live Site: [https://angsukan.github.io/portfolio-website/](https://angsukan.github.io/portfolio-website/)

*Note: Git credential helper configured and tested.*

---

## 🎨 Design & Aesthetic
* **Typography**: `JetBrains Mono` from Google Fonts for a sleek developer/analyst feel.
* **Theme**: Cyberpunk-inspired sleek dark mode with glassmorphism card styling and neon-glow accents:
  * **Accent Primary**: `#00f2fe` (cyan neon gradient)
  * **Accent Secondary**: `#4facfe` (blue gradient)
* **Animations**: Hover effects, responsive layout grid, and smooth interactive elements.

---

## 🛠️ Tech Stack
* **HTML5**: Semantic web structure.
* **CSS3**: Variables, custom layout system, and modern styling animations.
* **JavaScript (ES6+)**: Interactive project filtering, dynamic SQL/Python CLI simulation widgets, and page behavior.

---

## 📂 Key Features
* **Hero CLI Terminal**: Visual script terminal simulation executing SQL and Python data queries.
* **Interactive Dashboard Widget**: A dynamic live-stats component displaying analyst metrics.
* **Project Showcase**: A categorizable grid featuring key data analysis projects:
  1. *E-Commerce Customer Segmentation* (RFM Analysis & K-Means Clustering)
  2. *Automated ETL Pipeline & Live Dashboard* (Python, SQL, PowerBI/Tableau)
  3. *Financial Market Forecasting* (R, Time Series Analysis)
* **Contact & Socials**: Connected links to LinkedIn, GitHub, Email, and a contact gateway.
